# Курс «Тазовое дно. Реновация женского тела»

## Описание для лендинга

Онлайн-курс по восстановлению и укреплению мышц тазового дна. Классические практики йоги, адаптированные для женского здоровья.

## Для кого

- Женщины после родов
- При подготовке к беременности
- При симптомах ослабления мышц тазового дна
- Для профилактики

## Формат

- 21 день
- Видео-уроки
- Домашние задания с обратной связью
- Поддержка куратора

## Результаты

- Понимание анатомии и механики
- Регулярная практика
- Улучшение тонуса мышц
